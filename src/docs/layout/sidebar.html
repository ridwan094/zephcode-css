Sidebar

The sidebar component provides a vertical navigation layout for your application. It's highly customizable and responsive.

Usage

Add the zeph-sidebar class to create a sidebar layout. Inside, use zeph-sidebar-nav and zeph-sidebar-item for structured navigation.

Examples

Basic Sidebar

<aside class="zeph-sidebar">
    <nav class="zeph-sidebar-nav">
        <ul>
            <li class="zeph-sidebar-item"><a href="#" class="zeph-sidebar-link">Home</a></li>
            <li class="zeph-sidebar-item"><a href="#" class="zeph-sidebar-link">About</a></li>
        </ul>
    </nav>
</aside>

Customization

Use classes like zeph-sidebar-dark for a dark-themed sidebar or zeph-sidebar-collapsed for a compact view.
